# NDC-Counter

This model estimates the GHG emissions associated with the Nationally Determined Contributions (NDC) in the Paris Agreement.

It has been used in the following paper: Benveniste et al., 2018, "Impacts of nationally determined contributions on 2030 global greenhouse gas emissions: uncertainty analysis and distribution of emissions", Environmental Research Letters.
The article is open access and can be downloaded at http://iopscience.iop.org/article/10.1088/1748-9326/aaa0b9/meta
DOI: 10.1088/1748-9326/aaa0b9

Please quote this paper if you re-use the model.
